## shell

### `$_`

```sh
mkdir hoge
cd $_ # cd hoge と同じ
```

前回の引数として使える。


## ruby

### `_`（アンダースコア）

```rb
[1] pry(main)> Time.now
=> 2020-12-15 19:45:12.202559 +0900
[2] pry(main)> _
=> 2020-12-15 19:45:12.202559 +0900

[5] pry(main)> a = Time.now
=> 2020-12-15 19:45:33.885388 +0900
[6] pry(main)> _
=> 2020-12-15 19:45:33.885388 +0900
[7] pry(main)> _.object_id
=> 220
[8] pry(main)> a.object_id
=> 220
```

irbとかpryで前回の実行結果を取得できる。


### ``

```rb
names = %w[aaa bbb ccc]
names.each do |name|
  `mkdir #{name}`
ednd
```

``で囲めばコマンドを実行できる。

## 決意のあらわれ

### 全般

| ショートカット | できること | 習得した |
| ---- | ---- | ---- |
| `s`, `/` | 検索バーにフォーカス |
| `g→c` | 「Code」ページ |✔
| `g→i` | 「Issues」ページ |
| `g→p` | 「Pull Requests」ページ |
| `g→b` | 「Projects」ページ |
| `g→w` | 「Wiki」ページ |


### ファイルを開いた状態で

| ショートカット | できること | 習得した |
| ---- | ---- | ---- |
| `e` | 編集ページを開く |
| `t` | ファイルファインダーに飛ぶ |✔
| `w` | 別ブランチに |
| `l` | 行を検索 |
| `y` | コミットのURLになる |


### 「Issues」または「Pull Requests」リストで

| ショートカット | できること | 習得した |
| ---- | ---- | ---- |
| `c` | 新しく作成する |
| `a` | アサイン者で検索 |✔
| `l` | ラベルで検索 |
